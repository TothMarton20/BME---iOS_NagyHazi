# Expense Tracker

A native iOS and iPadOS expense tracking app that helps users manage their income and expenses through a modern, intuitive interface.

Built with **SwiftUI** and **SwiftData**, the app provides real-time visual insights using charts and dashboard components. It supports multiple currencies with automatic conversion and includes home screen widgets for quick balance overview.

## Features
- Track income and expenses by category  
- Interactive charts and financial insights  
- Multi-currency support (HUF, EUR, USD)  
- Receipt image attachments  
- Recurring transactions  
- iOS Home Screen widget support  

## Tech Stack
- SwiftUI  
- SwiftData  
- WidgetKit  
- Swift Charts  

## Goal
To demonstrate modern Apple technologies in a responsive, user-friendly financial application.
